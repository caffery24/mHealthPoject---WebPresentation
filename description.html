Our Project:

Many diseases are associate with eating habit. Among all the diseases, obesity has become one of the main challenges of people’s daily live. Thus, understanding an individual’s dietary behavior would give us better sense to improve eating habit. This project aims to sense the accelerate from X, Y, Z and gyro from X, Y, Z using Microsoft band to detecting hand to mouth gesture.

Four different gestures (eating burger with hand, drinking soup with spoon, eating noodles with folk and eating steak with knife) can be detected and then recognized by analyzing data using machine learning in the background. Besides, a proximity neck sensor is also applied to detect the frequency of corresponding eating gesture.

As the result of all the hand-to-mouth gesture detecting system, users could get feedback of their eating types and their eating frequency.
