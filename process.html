Timeline

  • Problem defined

    The primary goal of this project is to “recognize different eating gestures”. Our secondary goal, if time lets, is to detect the frequency of corresponding gestures and providing helpful feedbacks to the users. Different gestures are as follow: Eating with fork; Eating with spoon; Eating with Chinese sticks; Eating by hand;

  • HardWare Selection

    We chose two different sensors to collect the data we need. One is Microsoft Band 2, because it has embedded 3-axis accerleration sensor and 3-axis gyrometer which can give us accerleration data and gyro data. And we also use a proximity sensor Infrared Proximity Sensor Short Range - Sharp GP2Y0A41SK0F which can give us the distance between neck and wrist.

  • Plan of the future implementation

    Our project has 2 steps --- data collection and data analysis. Within the first part, we use a smart phone to collect the data send form the sensor and upload them to the database. Withink the second part, we visulize the data and use machine learning model to classify different data corresponding to different gesture.

  • Microsoft Band and proximity data collection

    We use the SDK provided by Microsoft to receive the sensor data from the Microsoft Band. After the smart phone has received the data, it uploads all the database to the specific database in real time.And we connect a multiprocessor to the proximity sensor which can directly complete AD change and upload the data to the database.

  • Data visulization

    The data visulazition is used to visualized data collected from Microsoft Band, which is accerleration and gyro data in three coordinates. We use MatLAB to visualize the data. For each data, we plot them in time zone.

  • Machine Learning

    First, we perform windowing on the obtained time series data from previous step. The main purpose of windowing is to consider the temporal sequencing between samples.The next step is modeling the windowed samples. We study different modeling technique to find the most appropriate one for our purpose.

  • Experiment and Result

    We have collect 7 participants' data as our trainning data. And we have also test 2 participants. The final accuracy can reach 80%.
